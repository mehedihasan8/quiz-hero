Difference map(), forEach(), filter() and find()

map, forEach, filter and find This is the array method. they each have a different purpose and can be used
1. The function of map is to return each element by throwing a loop with array.
2. The forEach function throws a loop but does not return anything.
3. find function that work find the first element that meets a condition.
4. filter function select all element that meets a condition and return with array.
